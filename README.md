## 从美国汇款到中国，具体怎么操作

很多人最关心的就是怎么往国内汇款。Remitly支持三种收款方式：银行账户、支付宝、微信。下面详细说说每种方式的操作流程。

**汇款到银行账户：**

这是最传统的方式。你需要提供收款人的姓名（必须和银行卡上的完全一致）、银行卡号、开户行信息等。一般来说，钱会直接存入收款人的银行账户，到账后会收到银行的短信通知。

需要注意的是，填写收款人信息时一定要仔细核对，特别是姓名的拼音和银行卡号，错一个字母都可能导致汇款失败。

**汇款到支付宝：**

这个功能特别方便。收款人不需要事先做什么设置，只要有支付宝账号就行。

汇款时，你需要输入收款人的支付宝绑定手机号和姓名。支付宝收到汇款后，会发一条短信通知，提示登录支付宝查看"跨境汇款"。收款人需要在48小时内打开支付宝，按照提示绑定一张银行卡，钱就会自动存入这张卡。

整个过程其实挺简单的，就是记得提醒收款人及时查看短信和支付宝通知，别错过48小时的绑定时限。

**汇款到微信：**

微信收款的流程和支付宝类似。收款人会收到一条带有微信链接的短信，点击链接后会打开"微汇款"小程序，按照提示授权绑定一张银行卡就行了。

同样的，这个授权也有48小时的时限，过期了就会比较麻烦。所以汇款前最好先跟家人说一声，让他们注意查收短信。
